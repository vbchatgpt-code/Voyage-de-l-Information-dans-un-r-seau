Découvre comment les données circulent à travers différents médias — Niveau 5e

Objectifs d'apprentissage

  • Comprendre que l'information numérique est codée en binaire (0 et 1).
  • Distinguer les supports de transmission : ondes radio, lumière, électricité.
  • Relier un même message à des signaux différents.
  • Différencier filaire / sans fil.
  • Utiliser un vocabulaire technique de base (bit, octet, routeur, fibre, RJ45).

Information

Toutes les informations numériques sont représentées par des zéros et des uns. C'est le code binaire. Un bit vaut 0 ou 1 ; 8 bits = 1 octet.
Le savais‑tu ? Une photo de 10 mégapixels contient environ 80 millions de 0 et de 1 !
